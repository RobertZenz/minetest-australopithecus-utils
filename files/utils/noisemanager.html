NoiseManager:get_noise_map (octaves, persistence, scale, spreadx, spready, spreadz, eased)
Creates a PerlinMap object with the given values.

Parameters

  • octaves: The count of octaves.
  • persistence: The persistence values.
  • scale: The scale.
  • spreadx: The spread of the x axis.
  • spready: Optional. The spread of the y axis, defaults to spreadx.
  • spreadz: Optional. The spread of the z axis, defaults to spreadx.
  • eased: Optional. If to use eased noise, defaults to false.

The new PerlinMap object.
