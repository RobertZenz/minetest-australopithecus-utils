Functions

get (name, default_value, cast_function)
Gets a value from the configuration as is.

Parameters:

  • name The name of the value to get.
  • default_value Optional. The default_value to return if the value is not set in the configuration. Can be nil.
  • cast_function Optional. The function to apply to the value that is read, for example to convert it into a different type. If this function returns nil, the default value IS returned. Can be nil.

Returns:

    The value from the configuration with the given name. If the read value is nil, the default value is returned or nil.

Parameters:

Returns:

    The boolean with the given name, or the default value if it is nil, or nil.
